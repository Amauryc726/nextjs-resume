# Next.js Resume Page

This project is a resume webpage built using Next.js and Tailwind CSS.

## How to Run

1. Install dependencies:
npm install

2. Start the development server:
npm run dev

3. Open in browser:
http://localhost:3000

## Features

- Resume sections (Summary, Education, Experience, Skills)
- Styled using Tailwind CSS
- Built with Next.js App Router

## Notes

- Simple and clean layout
- No major issues
