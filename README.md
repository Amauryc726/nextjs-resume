# Next.js Resume

## Description
This project is a professional resume website built with Next.js. It was created to present my skills, experience, and coursework in a cleaner and more organized way. The goal of this project is to create a digital resume that can be shared for internships, job applications, and interviews.

## Technologies Used
- Next.js
- JavaScript
- HTML
- CSS
- GitHub

## How to Run the Project
1. Clone or download the repository.
2. Open the project in Visual Studio Code.
3. Install dependencies using `npm install`.
4. Run the project using `npm run dev`.
5. Open the localhost link in a browser.

## What I Learned
While building this project, I learned how to organize a web project, work with a modern JavaScript framework, and present information in a professional format. I also learned how important it is to make a GitHub repository clear, readable, and interview-ready.
